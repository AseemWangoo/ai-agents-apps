## Things required

- Ollama on your machine with LLMs `gpt-oss:20b` and `mxbai-embed-large`

### Commands

- `python3 -m venv .venv`
- `source .venv/bin/activate`
- `pip install -r requirements.txt`

### Run

- `python main.py`

### In case of errors run

- `pip install -U <your_package>`
